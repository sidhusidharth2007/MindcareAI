The application needs an API key to talk to the AI. Follow these steps to set it up:

1.  Go to your repository **Settings** tab.
2.  On the left sidebar, click **Secrets and variables** > **Actions**.
3.  Click the green **New repository secret** button.
4.  Name: `API_KEY`
5.  Value: (Paste your Gemini API key from Google AI Studio)
6.  Click **Add secret**.
7.  Go to the **Actions** tab and **Re-run** the latest deployment job.
